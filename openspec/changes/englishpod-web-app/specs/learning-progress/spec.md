## ADDED Requirements

### Requirement: 学习进度持久化
系统 SHALL 记录每课的播放位置、完成状态与累计学习时长，并在重新打开时恢复。

#### Scenario: 恢复播放进度
- **WHEN** 用户重新打开此前未完成的课程
- **THEN** 音频定位到上次播放位置，状态显示为进行中

#### Scenario: 标记完成
- **WHEN** 音频播放完毕或用户手动标记
- **THEN** 该课状态变为已完成并记录完成时间

### Requirement: 学习仪表盘
系统 SHALL 在首页展示已完成课程数/365、本周学习天数、累计学习时长、生词总量与今日待复习数。

#### Scenario: 查看仪表盘
- **WHEN** 用户进入首页
- **THEN** 显示上述核心统计数据与「继续学习」卡片

### Requirement: 连续学习热力图
系统 SHALL 以热力图展示每日学习记录。

#### Scenario: 渲染热力图
- **WHEN** 用户查看仪表盘
- **THEN** 按每日学习记录渲染类似 GitHub 贡献图的热力图
